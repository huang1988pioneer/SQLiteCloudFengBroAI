# Triage Labels

The repo uses the default mattpocock/skills triage vocabulary:

| Role | Label |
| --- | --- |
| Maintainer needs to evaluate | `needs-triage` |
| Waiting on reporter | `needs-info` |
| Fully specified and agent-ready | `ready-for-agent` |
| Needs human implementation | `ready-for-human` |
| Will not be actioned | `wontfix` |

Use the exact label strings above unless the repository label policy changes.
